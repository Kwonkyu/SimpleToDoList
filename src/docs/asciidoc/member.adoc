include::{snippets}/MemberController/readMemberInfo/request-headers.adoc[HTTP Request Headers of Member Information API]

=== HTTP Response
.HTTP 응답
include::{snippets}/MemberController/readMemberInfo/response-body.adoc[HTTP Response of Member Information API]

.응답 파라미터
include::{snippets}/MemberController/readMemberInfo/response-fields.adoc[HTTP Response Payload of Member Information API]

팀(teams) 항목에는 팀의 식별자, 팀장의 아이디 및 이름, 팀의 이름, 잠금 상태 등 팀을 식별할 수 있는 값들과 팀의 할 일 리스트의 정보가 주어집니다. 할 일 리스트(todoLists) 항목에는 할 일 리스트의 식별자와 이름이 주어집니다.

기본적인 회원의 정보와 현재 회원이 어떤 팀에 가입되었는지, 해당 팀에는 어떤 할 일 리스트들이 있는지 팀과 할 일 리스트를 식별할 수 있는 정보를 제공합니다.

필요 시 팀의 할 일 리스트나 팀의 식별자를 이용하여 추가적인 API 를 활용할 수 있습니다.

* 팀 응답 항목에 대한 문서는 <<team.adoc#_team_information, 이곳>>을 참조하십시오.
* 할 일 리스트 응답 항목에 대한 문서는 <<todolist.adoc#_todolist_information, 이곳>>을 참조하십시오.
* 할 일 응답 항목에 대한 문서는 <<todo.adoc#_todo_information, 이곳>>을 참조하십시오.
