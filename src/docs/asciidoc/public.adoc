NOTE: 모든 요청, 응답의 바디(페이로드)는 JSON 으로 구성되어 있습니다.

== 로그인
NOTE: 인증이 필요하지 않은 *공개된*  API 입니다.

가입된 회원의 아이디와 비밀번호로 로그인하는 기능입니다. 성공 시 JWT 값이 *token* 필드로 반환됩니다.

이후 토큰은 제한된 API 요청 시 Authorization 헤더에 Bearer 토큰으로 담아서 전송해야 합니다.
